Checker Code
  thisSubtraction
Description
  Suspicious pointer subtraction.
Severity
Minor
Detector / Bug Pattern
Suspicious pointer subtraction.
More Information
This message indicates an unintended side effect to the pointer arithmetic (or array indexing) found in an expression.
Bad Code 
int main()
{
     int const anila=10;
     int *ptr=&anila;
     clrscr();
     printf("%d",*ptr);
     getch();
     return 0;
}
Good Code 
int main()
{
     int const anila=10;
     int *ptr=&anila;
     *ptr = 100; // --> This should not be done since you declare anila as const
     printf("%d",*ptr);
     return 0;
}
